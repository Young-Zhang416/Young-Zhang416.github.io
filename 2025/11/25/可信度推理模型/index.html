可信度的概念

可信度是指人们根据以往经验对某个事物或现象为真的程度的一个判断，或者说是人们对某个事物或现象为真的相信程度。例如，老师昨天没来上课，理由是头疼。就此理由，只有以下两种可能：一是真的头疼了，理由为真；二是没有头疼，理由为假。但就听话人而言，因不能确切知道，就只能某种程度上相信，即可信度。可信度具有一定的主观性，较难把握。但对某一特定领域，让该领域专家给出可信度还是可行的。

可信度推理模型

表示形式
在C-F模型中，知识使用产生式规则表示，其一般形式为：

IF E  THEN H (CF(H,E))IF \ E\ \ THEN\ H\ (CF(H,E)) IF E  THEN H (CF(H,E))

其中，E是知识的前提条件，H是知识的结论；CF(H,E)是知识的可信度。CF是知识的静态强度，取值范围为[−1,1][-1,1][−1,1]，表示当E为真时，证据对H的支持程度，其值越大，支持程度越大。

在C-F模型中，CF(H,E)被定义为：

CF(H,E)=MB(H,E)−MD(H,E)CF(H,E)=MB(H,E)-MD(H,E) CF(H,E)=MB(H,E)−MD(H,E)

式中MB为信任增长度，有：

可信度的性质

(1) 互斥性

核心概念：对同一证据E，它不可能既增加对假设H的信任程度，又同时增加对H的不信任程度。这体现了信任增长度MB（Measure Belief）与不信任增长度MD（Measure Disbelief）之间的互斥关系。

数学表达：

{当 MB(H,E)>0 时,MD(H,E)=0当 MD(H,E)>0 时,MB(H,E)=0\begin{align*} \begin{cases} \text{当 } MB(H, E) > 0 \text{ 时}, & MD(H, E) = 0 \\ \text{当 } MD(H, E) > 0 \text{ 时}, & MB(H, E) = 0 \end{cases} \end{align*} {当 MB(H,E)>0 时,当 MD(H,E)>0 时,​MD(H,E)=0MB(H,E)=0​​

实际意义：这一性质确保了证据对假设的影响是单向的——要么支持，要么反对，不会同时存在矛盾的影响。

(2) 值域范围

可信度因子的取值范围具有明确的上下界[0,1][0,1][0,1]

值域解释：

  • MB(H,E)：信任增长度的取值范围为[0,1][0,1][0,1]，表示从完全不信任到完全信任的程度

  • MD(H,E)：不信任增长度的取值范围同样为[0,1][0,1][0,1]，表示从完全信任到完全不信任的程度

  • CF(H,E)：综合可信度的取值范围为[−1,1][-1,1][−1,1]，其中负值表示不信任，正值表示信任

(3) 典型值分析

可信度因子在边界值处具有明确的概率意义：

当CF(H,E) = 1时

  • 概率意义：P(H∣E)=1P(H|E) = 1P(H∣E)=1

  • 含义解释：证据E的出现使假设H为真

  • 分量状态：MB(H,E)=1MB(H,E) = 1MB(H,E)=1，MD(H,E)=0MD(H,E) = 0MD(H,E)=0

当CF(H,E) = -1时

  • 概率意义：P(H∣E)=0P(H|E) = 0P(H∣E)=0

  • 含义解释：证据E的出现使假设H为假

  • 分量状态：MB(H,E)=0MB(H,E) = 0MB(H,E)=0，MD(H,E)=1MD(H,E) = 1MD(H,E)=1

当CF(H,E) = 0时

  • 分量状态：MB(H,E)=0MB(H,E) = 0MB(H,E)=0，MD(H,E)=0MD(H,E) = 0MD(H,E)=0

  • 双重含义：

    • MB=0MB=0MB=0：证据EEE不证实假设HHH

    • MD=0MD=0MD=0：证据EEE不否认假设HHH

(4) 对偶性质：H与¬H的关系

根据MB、MD的定义及概率公理，可以推导出重要的对偶关系：

信任增长度的对偶性：

MD(¬H,E)=MB(H,E)MD(¬H,E)=MB(H,E) MD(¬H,E)=MB(H,E)

可信度之和的性质：

根据CF的定义和MB、MD的互斥性：

CF(H,E)+CF(¬H,E)​=0​CF(H,E)+CF(¬H,E)​=0​ CF(H,E)+CF(¬H,E)​=0​

重要推论：

  1. 对HHH的信任增长度等于对非HHH的不信任增长度

  2. 对HHH的可信度与非HHH的可信度之和等于0

  3. 关键区别：可信度不是概率，不满足概率的公理条件：

    • P(H)+P(¬H)=1P(H) + P(¬H) = 1P(H)+P(¬H)=1

    • 0≤P(H),P(¬H)≤10 ≤ P(H), P(¬H) ≤ 10≤P(H),P(¬H)≤1

(5) 多结论约束条件

当同一证据EEE支持多个互斥的结论Hi(i=1,2,...,n)H_i(i=1,2,...,n)Hi​(i=1,2,...,n)时，存在重要的约束条件：

约束不等式：

∑inCF(Hi​,E)≤1\sum_{i}^nCF(H_i​,E)≤1 i∑n​CF(Hi​​,E)≤1

实际应用意义：这一性质在专家系统知识库建设中具有重要的质量控制作用。如果发现专家给出的知识出现如下的非法情况：

CF(H1​,E)=0.7,CF(H2​,E)=0.4CF(H1​,E)=0.7,CF(H2​,E)=0.4CF(H1​,E)=0.7,CF(H2​,E)=0.4

由于0.7+0.4=1.1>10.7 + 0.4 = 1.1 > 10.7+0.4=1.1>1，这违反了可信度的数学约束，需要进行调整或规范化处理。

工程实践：在实际专家系统开发中，这一性质为知识工程师提供了验证规则库合理性的重要数学依据，确保推理结果的可靠性和一致性。

用CF(E)CF(E)CF(E)表示证据的不确定性，如果：

  • CF=−1CF=-1CF=−1，证据为假
  • CF=0CF=0CF=0，对证据一无所知

不确定性的更新

可信度推理的核心思想是从初始的不确定证据出发，结合相关不确定性知识，逐步推出最终结论，并计算其可信度。每次推理过程中，都需要根据证据的不确定性（CF(E)）和规则的可信度（CF(H,E)），来更新结论H的可信度。

基本公式如下：

CF(H)=CF(H,E)×max⁡{0,CF(E)}CF(H) = CF(H, E) \times \max\{0, CF(E)\} CF(H)=CF(H,E)×max{0,CF(E)}

  • 若 CF(E)<0CF(E) < 0CF(E)<0，则 CF(H)=0CF(H) = 0CF(H)=0，表示证据E为假时，不考虑其对结论H的影响。
  • 若 CF(E)=1CF(E) = 1CF(E)=1，则 CF(H)=CF(H,E)CF(H) = CF(H, E)CF(H)=CF(H,E)，即此时规则强度 CF(H,E)CF(H,E)CF(H,E) 实际上代表了当E为真时，H的可信度。

这一过程体现了系统如何根据证据的支持程度动态调整结论的可信度，尤其适用于证据不确定或部分支持的情况。

2. 结论不确定性的合成

当多个独立的知识前提支持同一个结论，且各自的可信度不同时，需要对这些信息进行综合，以得出结论的综合可信度。

假设存在两条知识：

  • IF E1E_1E1​ THEN HHH（可信度为 CF(H,E1)CF(H, E_1)CF(H,E1​)）
  • IF E2E_2E2​ THEN HHH（可信度为 CF(H,E2)CF(H, E_2)CF(H,E2​)）

则结论H的综合可信度可通过以下两步计算：

(1) 分别计算每条知识对结论的贡献：

CF1(H)=CF(H,E1)×max⁡0,CF(E1) CF2(H)=CF(H,E2)×max⁡0,CF(E2)CF_1(H) = CF(H, E_1) \times \max{0, CF(E_1)} \ CF_2(H) = CF(H, E_2) \times \max{0, CF(E_2)} CF1​(H)=CF(H,E1​)×max0,CF(E1​) CF2​(H)=CF(H,E2​)×max0,CF(E2​)

(2) 合成两条证据对H的综合可信度：

CF(H)={CF1(H)+CF2(H)−CF1(H)⋅CF2(H),若 CF1(H)≥0 且 CF2(H)≥0CF1(H)+CF2(H)+CF1(H)⋅CF2(H),若 CF1(H)<0 且 CF2(H)<0CF1(H)+CF2(H)1−min⁡{∣CF1(H)∣,∣CF2(H)∣},若 CF1(H) 与 CF2(H) 异号CF(H) = \begin{cases} CF_1(H) + CF_2(H) - CF_1(H) \cdot CF_2(H), & \text{若 } CF_1(H) \geq 0 \text{ 且 } CF_2(H) \geq 0 \\ CF_1(H) + CF_2(H) + CF_1(H) \cdot CF_2(H), & \text{若 } CF_1(H) < 0 \text{ 且 } CF_2(H) < 0 \\ \dfrac{CF_1(H) + CF_2(H)}{1 - \min\left\{ |CF_1(H)|, |CF_2(H)| \right\}}, & \text{若 } CF_1(H) \text{ 与 } CF_2(H) \text{ 异号} \end{cases} CF(H)=⎩⎨⎧​CF1​(H)+CF2​(H)−CF1​(H)⋅CF2​(H),CF1​(H)+CF2​(H)+CF1​(H)⋅CF2​(H),1−min{∣CF1​(H)∣,∣CF2​(H)∣}CF1​(H)+CF2​(H)​,​若 CF1​(H)≥0 且 CF2​(H)≥0若 CF1​(H)<0 且 CF2​(H)<0若 CF1​(H) 与 CF2​(H) 异号​

该公式确保了在不同证据支持方向一致或冲突时，能够合理地融合信息，避免过度放大或抵消。
